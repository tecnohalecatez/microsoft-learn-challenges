1. ¿Qué dos características de Azure AI Services se pueden usar para habilitar texto a texto y conversión de voz en texto entre varios idiomas? Cada respuesta correcta presenta parte de la solución.
    - el servicio voz
    - el servicio Translator

    El servicio Voz de Azure AI se puede usar para generar audio hablado a partir de un origen de texto para la traducción de texto a voz. El servicio Azure AI Translator admite directamente la traducción de texto a texto en más de 60 idiomas. La extracción de frases clave, Conversational Language Understanding y la detección de idioma no se usan para la traducción de idioma para la traducción de texto a texto y conversión de voz a texto.


1. ¿Qué característica del servicio Azure AI para lenguaje le permite analizar artículos escritos para extraer información y conceptos, como personas y ubicaciones, con fines de clasificación?
    - reconocimiento de entidades con nombre

    El reconocimiento de entidades con nombre puede identificar y clasificar entidades en texto no estructurado, como personas, lugares, organizaciones y cantidades, y es adecuado para admitir el desarrollo de un sistema de recomendaciones de artículo. La extracción de frases clave, Content Moderator y la característica PII no son adecuadas para las tareas de reconocimiento de entidades para crear un sistema de recomendación.


1. ¿Qué tipo de traducción admite el servicio Azure AI Translator?
    - texto a texto

    El servicio Traductor de Azure AI admite la traducción de texto a texto, pero no admite la conversión de voz a texto, texto a voz o conversión de voz a voz.


1. ¿Qué tres características son elementos del servicio Azure AI Speech? Cada respuesta correcta presenta una solución completa.
    - identificación del idioma
    - reconocimiento del hablante
    - asistentes de voz

    La identificación del idioma, el reconocimiento del hablante y los asistentes de voz son todos los elementos del servicio De voz de Azure AI. La traducción de texto y la traducción de documentos forman parte del servicio Translator.


1. ¿Qué característica del servicio Azure AI Translator solo está disponible para Custom Translator?
    - entrenamiento de modelos con un diccionario

    El entrenamiento de modelos con un diccionario se puede usar con Custom Translator cuando no tiene suficientes oraciones paralelas para cumplir los 10 000 requisitos mínimos. El modelo resultante normalmente completará el entrenamiento mucho más rápido que con el entrenamiento completo y usará los modelos de línea base para la traducción junto con los diccionarios que ha agregado.


1. Al usar azure AI Service for Language, ¿qué debe usar para proporcionar más información en línea sobre las entidades extraídas de un texto?
    - vinculación de entidad

    Entity Linking identifica y desambigua la identidad de las entidades que se encuentran en un texto. La extracción de frases clave no se usa para extraer entidades y se usa en su lugar para extraer frases clave para identificar los conceptos principales de un texto. El reconocimiento de entidades con nombre no puede proporcionar un vínculo para cada entidad para ver más información. La traducción de texto forma parte del servicio Azure AI Translator.
